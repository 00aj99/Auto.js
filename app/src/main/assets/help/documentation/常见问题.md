### 如何重复做某些操作n次
使用循环语句实现实现。例如每个1秒执行一个操作，总共执行20次的代码如下:
```
for(var i = 1; i <= 20; i++){
    //这里写要执行的操作
    sleep(1000);
}
```

### 某些有文字的按键点击无效怎么办
例如QQ下方的"联系人"按钮，用click("联系人")是无效的。这是因为这些按钮上的文字并不是真正的文字，而是一个图标。参见[如何点击图标按钮](#如何点击图标按钮)。
